ZTBaliston

  • version

    ZTBalistonKit SDK version

  • shared

    Shared instance

  • dfuDelegate

    DFU delegate

  • connectedDevice

    Connected ZTDevice. Connection is done using ZTCoreKit.

  • isActivityRunning

    Activity is runnign on modules If nil is returned ZTBaliston.shared.readGeneralState() should be called to update it

  • lastSerialNumber

    Last serial number of device connected

  • currentActivity

    Current activity instance

  • currentDfuState

    Current DFU state for background DFU. If state is .readyForCopy - shoes can be restarted to apply new firmware. Also firmware will be updated automatically after attaching/detaching power cable. Use method ZTBaliston.shared.reboot(completion:) to reboot shoes
